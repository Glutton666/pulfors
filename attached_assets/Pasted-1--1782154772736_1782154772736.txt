1단계 — 새 악보 시작
지금은 새 악보를 만들 때 제목·작곡가·박자·조성을 한꺼번에 입력해야 해요. 앞으로는 제목만 입력하면 바로 에디터로 들어가고 사용자 이름으로 제작자 자동으로 생김. 박자·BPM·조성은 안에서 마디별로 조정해요.(어떻게 조정하게 할지 설명)

2단계 — 악보 작성 (바모드 스타일)
에디터 화면 하단에 드로어 패널이 항상 있어요. 이 패널이 바모드의 마디 편집 영역과 같은 역할을 해요.

작성 순서:

현재 마디에 음표를 그린다
드로어에서 이 마디의 박자·BPM을 확인하거나 조정한다
"다음 마디 ▶" 버튼을 누르면 드로어가 위로 올라오며 다음 마디로 이동 (바모드에서 마디 추가하는 느낌)
완성된 마디를 탭하면 언제든 다시 불러와 수정 가능(오버뷰 버튼이 있어 현재까지 작성한 악보 전부 보여주고 거기서 선택해서 편집화면으로 돌아가는 느낌)
→ 마디를 다 채워도 자동으로 넘어가지 않아요. 사용자가 명시적으로 넘겨야 해요.(다 채운다는 개념이 없음. 스케치북에 사용자가 스케치를 하는 느낌으로 작성하는 거라 형식이 다른데, 현재 어떻게 악보를 작성하는지에 대해서 내가 말하는 것과 다르게 알고 있는것 같아서 현재 인지중이 작성방법 알려주기 바람)

3단계 — 음표 입력 (더 정확하게)
손가락을 오선지에 올리면 반투명 고스트 음표가 실시간으로 손가락을 따라와요
고스트 음표 옆에 현재 음이름 (예: C4, G#3)과 인접 칸 표시가 함께 보여서 어디에 놓이는지 명확하게 알 수 있어요
손가락을 떼는 순간 그 위치에 음표가 확정돼요.(음표 확정되는 위치가 프로그램에 의해서 조정되면 안됨. 음표끼리 겹쳐도 겹치는대로 두는 느낌으로 되어야함. 연음계열에 대해서 어떻게 할지 계획 설명 필요)
악보 기호 자동 삽입:

박자나 조성이 바뀌는 마디 경계엔 겹세로줄이 자동으로 그려져요
악보 마지막에는 끝세로줄이 자동으로 붙어요
4단계 — 악기 기호 (스티키 방식으로 수정)
지금은 스타카토·악센트 같은 기호가 켜지고 꺼지는 방식이라 이상하게 동작해요. 앞으로는:

기호를 팔레트에서 선택하면 그 이후 놓는 음표마다 자동으로 기호가 붙어요 (sticky 모드)
이미 기호가 붙은 음표를 탭하면 그 기호만 제거돼요
5단계 — 마디 오버뷰
에디터 상단 또는 버튼으로 오버뷰 모드로 전환하면, 모든 마디가 작게 나열돼요.

바모드 프리셋이 연결된 마디는 색으로 구분 표시
마디를 탭하면 그 마디로 이동해 편집 가능 (이전으로 돌아오기)
6단계 — 바모드 프리셋 마디 범위 지정
악보 작성이 끝나면, 마디를 드래그해 범위를 하이라이트로 선택할 수 있어요.(재생이 겹치지 않게 잘 조정되어야함)

예: 1-8마디 드래그 선택 → "바모드 프리셋 연결" 버튼 탭 → 연습장에서 프리셋 선택
연결된 범위는 색·라벨로 표시되고, 탭하면 다시 변경·해제 가능
이렇게 하면 같은 악보에서도 구간마다 다른 메트로놈 설정으로 연습할 수 있어요

이렇게 작성이 완료된 악보는 악모모드 첫화면 및 연습장에서 재생버튼으로 재생 및 불러와서 볼수 있음

Task #315 — 공유 + 노트모드 연동 (#314 이후)
악보 공유
에디터 상단 공유 버튼 → 두 가지 선택:

JSON 파일: 다른 기기에서 악보를 그대로 다시 열 수 있어요
이미지(PNG): 카톡·SNS 등으로 바로 공유
공유받은 .score.json 파일을 불러오면 에디터에서 바로 열려요.

노트모드 큐에서 악보 미리보기
노트모드 큐 아이템에 악보가 연결되어 있으면 사진 대신 악보 미리보기가 자동으로 표시돼요.

재생 중에 악보가 따라가는 단위를 설정할 수 있어요:(하이라이트는 미실시)

박 단위: 한 박씩 하이라이트(박단위는 삭제)
마디 단위: 현재 마디 강조
악구 단위: 설정한 마디 수(예: 4마디)씩 표시
재생 중 바모드 프리셋 자동 전환
재생 중에 마디가 바뀌면 연결된 바모드 프리셋의 BPM·박자·사운드가 자동으로 메트로놈에 적용돼요.(추가적으로 바모드 적용되어 있으면 메트로놈 재생은 미실시)

예: 1-8마디 재생 → 프리셋 A의 설정으로 재생
9마디 진입 → 프리셋 B로 자동 전환
